void Sculptor::writeOFF ( const char *  filename)

writeOFF

Parameters
filename

Essa função é responsável por escrever o arquivo OFF no disco.

O arquivo OFF deve conter as informações do formato OFF, seguidas das coordenadas de todos os vértices ativos e, por fim, as faces (triângulos) correspondentes a cada voxel ativo.
